# 🧬 X-Ray Vision Hand Portal UI

🚀 An advanced futuristic **X-Ray Hand Portal Interface** built using **MediaPipe, Computer Vision, and Canvas Rendering**.

This project creates a **real-time interactive portal** using hand gestures, where users can visualize a sci-fi style **neural X-ray body system** with particles, grids, and HUD elements.

---

## 🌐 Live Demo

👉 https://shivank302.github.io/xray-hand-portal-ui/

---

## ⚡ Features

✨ Real-time Hand Tracking (MediaPipe)
✨ Dual-hand gesture detection to activate portal
✨ Futuristic X-ray body visualization
✨ 10,000+ dynamic neural particles system
✨ Interactive portal with glowing edges
✨ Live HUD (Heads-Up Display) system
✨ Glitch effects & scan interruptions
✨ Energy particles & spark effects
✨ Fully responsive full-screen experience

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Canvas API
* MediaPipe Holistic

---

## 🎮 How It Works

1. Open the website
2. Allow camera access 🎥
3. Show both hands in front of camera
4. Bring fingers close 🤏
5. Portal activates 🌀
6. Move hands to control the X-ray visualization
